Setup

Proxmox setup steps

  1. Great installation video
  2. download iso
  3. Burn iso to a flashdrive
    1. linux: dd command
    2. Windows: rufus
    3. MacOS: Etcher
  4. Put flash drive into target machine
  5. Boot from the flash drive
    1. or using iLO
  6. Follow the prompts and fill in the information
    1. Note: i did have to configure my disks in some of the boot menus to make a single logical volume RAID 5 configuration
    2. The in the Proxmox installer i just selected ext4 for the file system
    3. Will use ZFS on Fafnir
  7. fill in network information
    1. Much of it can be gleaned from your router console
    2. ifconfig get default | grep gateway for the default gateway
    3. for MacOS to find DNS server just search for DNS in system preferences
  8. Once proxmox is finished bootstrapping and restarts the server, remove the flash media
  9. if screen loads to a console then you can just transition back to your main machine and use the web interface to finish.
  10. management console is at: https://#.#.#.#:8006 (replace # with valid ipv4 address)
  11. connection will be unsecure and that's okay, proceed anyhow
  12. login as root with the password you previously set.
  13. Ignore subscription popup, its FOSS unless you want enterprise subscription
