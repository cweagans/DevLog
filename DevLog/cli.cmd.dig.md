`D`omain `I`nternet `G`roper

Useful with [[n.protocol.dns]]

## Usage

```bash
dig www.google.com
```

### Specify a DNS server manually

This is useful for troubleshooting local DNS

```bash
dig @8.8.4.4 www.google.com
```
