### insert( index, subStr )

Inserts string `subStr` starting at `index`.

```cpp
// userText is "Goodbye"
userText.insert(0, "Well "); // Now "Well Goodbye" 
// userText is "Goodbye"
userText.insert(4, "---");   // Now "Good---bye"
```
