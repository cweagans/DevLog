## Proxmox setup steps

1. [download iso][1]
2. Burn iso to a flashdrive
   1. linux: `dd` command
   2. Windows: [rufus][2]
   3. MacOS: [Etcher][3]
3. Put flash drive into target machine
4. Boot from the flash drive

[1]: https://proxmox.com/en/downloads
[2]: https://rufus.ie/en/
[3]: https://www.balena.io/etcher/
